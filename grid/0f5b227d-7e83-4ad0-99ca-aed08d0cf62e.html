Description



In Emergency Call 112 Showroom, you have the opportunity to look at all currently done vehicles from Emergency Call 112 – The Fire Fighting Simulation 2 in detail for the first time.



The Emergency Call 112 Showroom offers you the possibility to design and implement your own textures and share them with the community via the [Steam Workshop].
Compare notes with other gamers and build the textures of your fire fighting station so that it is ready for the main game.

To edit the Texture we recommend to use Adobe Photoshop / GIMP / PAINT.NET - make sure to save the file as DXT1 DDS and Mipmaps (maybe require a plugin) and mirror the image vertical before export!
