Description

Major function


Fully functional VR web browser

It supports fluency browsing of various websites and video playing on-line. Browsers can be displayed on both VR environment and PC desktop simultaneously. You can watch all kinds of movie files during downloading and it also has File Transfer Protocol function. The latter allow users to continue the transmission from the place of breakage so it greatly promote work efficiency.
We create our own website: www.mermvr.com.
It pools more VR video resource sites, which makes it much more easier for users to find their favorite videos.

3D Cinema mode

The brand new3D Cinema mode, it makes you feel like you are in the huge theater of virtual reality and switch seat and watching angle smoothly.

Safeguard User privacy

You can choose whether to copy the PC terminal home and aboard simultaneously in settings. It effectively protects users privacy when they watch all kinds of films, which make them really own their own Virtual Theater.

Full record of media using habit

Mermaid VR has unique full record of media using habit, once each media file is played, all setting information such as panoramic view, 180 degrees, 3D, brightness, color, hyper sampling and so on can be automatically recorded independently. If the user replays, he just needs to click the play button and the panoramic video will present with original settings.

It can also play inside and outside subtitles+ Multi track video

Mermaid VR player has seamless connection with inside and outside subtitles+ Multi track video, so users can switch all kinds of tracks and subtitles as they want. One movie, various taste.
